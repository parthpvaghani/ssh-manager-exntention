# GitHub SSH User Switcher

Easily switch between multiple GitHub SSH identities and ensure correct commit authorship. Works in VSCode and Cursor.

## Features
- List all available GitHub SSH identities (from ~/.ssh/config and extension-managed)
- Mark and switch the active identity for GitHub with one click
- Add, edit, or remove GitHub SSH identities
- Ensures correct SSH key is used for GitHub operations
- Marketplace-ready, works in VSCode and Cursor

## Usage
1. Open the SSH Manager view from the Activity Bar.
2. Add or discover your GitHub SSH keys.
3. Right-click any identity and select "Set as Active for GitHub".
4. The active identity is marked and used for all GitHub SSH operations.
